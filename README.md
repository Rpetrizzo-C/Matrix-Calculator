# Matrix-Calculator

### This is a Matrix Calculator

  [![forthebadge](https://forthebadge.com/images/badges/powered-by-black-magic.svg)](https://forthebadge.com)

### What it can do?
 
  * Add Matrices
  * Multiplying Matrices by a Constant
  * Multiply Matrices
  * Transpose Matrix
  * Calculate Determinant
  * Inverse matrix
